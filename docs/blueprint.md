# **App Name**: Portfolio Pro

## Core Features:

- Project Showcase: Display professional projects with descriptions, images, and links.
- About Me Section: Detailed information about the user, including skills, experience, and education.
- Contact Form: A functional contact form that sends messages to the user's email.
- Skills Visualizer: Visually represent skills and proficiency levels using charts or graphs.
- AI-Powered Portfolio Review Tool: Use a tool to give personalized feedback on portfolio content, suggest improvements, and highlight strengths. 

## Style Guidelines:

- Primary color: Navy blue (#3F51B5) for professionalism and trust.
- Background color: Light gray (#EEEEEE) to provide a clean and modern backdrop.
- Accent color: Teal (#009688) to highlight important elements and call-to-action buttons.
- Body and headline font: 'Inter', a sans-serif font for a modern, neutral feel.
- Use a set of minimalist icons for skills and project categories.
- A clean, single-page layout with clear sections for easy navigation.
- Subtle transition animations when navigating between sections.
